REST API для роботи з коллекцією контактів

  • Для реєстрації нового користувача необхідно відправити POST - запрос на ендпоінт: /auth/register

                    в body {
                    "email": "ivandemid@gmail.com",
                    "password": "ts7G8jhShWoz9"
                }

    У відповіді на запит прийде URL-згенерованого аватара, після цього на вказану адресу прийде лист з посиланням, на яке необхідно перейти для верифікації користувача

  • Після реєстрації необхідно здійснити login користувача, для цього відправити POST - запрос на ендпоінт: /auth/login

                    в body {
                    "email": "ivandemid@gmail.com",
                    "password": "ts7G8jhShWoz9"
                }

    У відповіді на запит прийде токен, який необхідно вказувати у заголовку Authorisation

  • Для виходу користувача використати метод POST, ендпоінт /auth/logout

  • Для завантаження аватара з комп'ютера користувача потрібно використати метод PATCH, ендпоінт /users/avatars. В body вказати тип form-data, створити ключ із назвою avatar і типом file, вибрати файл зображення який необхідно вигрузити і відправити запрос. Після чого у відповіді прийде URL-нового зображення

Список ендпоінтів:

  • POST: /auth/register
  • POST: /auth/login
  • POST: /auth/logout; Authorization: "Bearer token"
  • PATCH: /users/avatars; (body: form-data; key: avatar )
  • GET: /users/current

  • GET: /api/contacts
  • GET: /api/contacts/:contactId
  • POST: /api/contacts
  • PATCH: /api/contacts/:contactId
  • DELETE: /api/contacts/:contactId

Query:

page, limit, sub ["free", "pro", "premium"]
